Projeto para nota da AV2

Orientações:

Esse projeto vale 8,0 (oito) pontos. Os outros 2,0 (dois) pontos serão obtidos com a entrega da atividade prática de JavaScript;

O site deverá ser desenvolvido individualmente. Assunto a ser desenvolvido é livre;

Ele deve ser composto por, no mínimo, 3 páginas html com as devidas navegações entre as páginas. Como exemplo: index.html, galeria.html, servicos.html;

O tema e layout a ser aplicado na construção do site deve ser apresentado à professora e aprovado por ela antes da entrega (até 15/06/2021

Obrigatória implementação de estruturas de pasta completas (nomenclatura, subpastas de arquivos etc.) além de uso de padrões HTML5 com a devida indentação do código;

Também é obrigatório: (1) uso de estilização e formatação por CSS inline, interno e/ou externo; e (2) script em Javascript de interação com usuário; (4) implementação de uma barra lateral à esquerda ou direita (tipo as propagandas que aparecem das pesquisas usando Google – conteúdo da barra depende do projeto);

Opcional: (4) Layout Responsivo; e (5) uso de CANVAS

Pode-se usar arquivo externo CSS ou JS se for escrito pelo aluno, caso seja identificado necessidade dele;

É PROIBIDO uso de arquivos externos do W3C (CSS e Javascript).

Prazo máximo de postagem (arquivo zipado no Google Classroom – obrigatório nome do aluno no nome do arquivo): 28/06/2020 às 11:00h (sem prorrogação).

ATENÇÃO: Não será permitido uso de geradores de código, templates e frameworks de qualquer tipo. A constatação do uso desses implicará em conceito Zero (0) no projeto.
